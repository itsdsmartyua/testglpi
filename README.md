# GLPI Plugin: TelegramBot

Minimal skeleton plugin for **GLPI 11.0.4**.

✅ Shows **Install/Uninstall** buttons  
✅ On install creates DB table `glpi_plugin_telegrambot_configs`  
✅ Has a basic config page (stub)

## Install

1. Copy to: `glpi/plugins/telegrambot`
2. GLPI UI: **Setup → Plugins** → **Install** → **Enable**
